Schedule multiple sale for your product at once in WooCommerce effortlessly, ensuring timely discounts and seamless promotions.

== Description ==

Multiple Sale Scheduler allows you to schedule multiple sales for your WooCommerce products with ease. Manage sales periods, apply discounts automatically, and increase customer engagement with planned promotions.

== Features ==

Schedule sales for individual or multiple products.

Set start and end dates for each sale.

Automatically apply sale prices during the scheduled period.

Easy-to-use interface within WooCommerce product settings.

Compatible with the latest WooCommerce and WordPress versions.

== Why Use Multiple Sale Scheduler? ==

Running multiple sales manually can be time-consuming and error-prone. Multiple Sale Scheduler streamlines the process by automating sales and ensuring they start and stop on time, helping you focus on growing your business.

== Installation ==

From the admin panel, Go to your WordPress Admin -> Plugins -> Add New. Search for Multiple Sale Scheduler. Install and Activate.

Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

Activate the plugin through the 'Plugins' screen in WordPress.

Navigate to product edit page in the WooCommerce to start scheduling sales.

In general product data tab you can see new field if product type is simple.
